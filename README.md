Globeflight Courier Shipping Module
===================================

*Globeflight Courier Shipping Module for WooCommerce, an eCommerce plugin for Wordpress.*

Documentation
-------------

Please refer to our documentation at http://globeflight.github.io/Globeflight-WooCommerce

This plugin makes use of our [Courier-Client](https://github.com/Globeflight/Courier-Client) which in turn makes use of our [Webservice](http://globeflight.co.za/woo/wsdocs/).

If you wish to submit any changes, simply fork this repository and submit a pull request.
